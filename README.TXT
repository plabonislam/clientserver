                                

                        Client-Server Chat Application

1. Firstly open Chat_Server.jar and click Start button to open server

2. Then open Chat_Client.jar to create one client.If you want to create more
client you need to open Chat_Client.jar several time

3.After opening client establish new connection by submitting username and 
password. You can also use anonymous Login option to establish the connection

4.Finally you are ready to send message among connected clients

5.You can also save and view message history
